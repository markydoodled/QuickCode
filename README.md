## Features

- iOS App
- iPadOS App
- macOS App
- Syntax Highlighting
- Editor Themes
- In-App File Metadata Viewer
- App Clip Support - [Click To Launch...](https://appclip.apple.com/id?p=com.MSJ.QuickCode.Clip)
- macOS Dock Menu Quick Actions
